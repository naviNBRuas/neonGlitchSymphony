## Installation
Ensure you have **Node.js** and **Bun** installed.

```sh
# Clone the repository
git clone https://github.com/naviNBRuas/neon-glitch-symphony.git
cd neon-glitch-symphony

# Install dependencies
bun install

# Start development server
bun run dev
```
